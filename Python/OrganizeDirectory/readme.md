# Organize Directory

Allows to organize a messy directory by categorizing files into folders.

#### This allows to organize the following file types and extensions:

- Documents:
  - '.pdf', '.rtf', '.txt', '.md', '.xlsx', '.doc'
- Audio:
  - '.m4a', 'm4b', '.mp3'
- Videos:
  - '.mov', '.avi', '.mp4', '.flv', '.mkv'
- Images:
  - '.jpg', '.jpeg', '.png', '.svg', '.gif'
- Python_files:
  - '.py', '.pyc'
- CPP_files:
  - '.cpp', '.hpp'
- Executables:
  - '.bat', '.exe'
